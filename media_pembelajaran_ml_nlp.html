Apa itu Machine Learning?

Machine Learning (ML) adalah cabang ilmu komputer yang memungkinkan mesin untuk belajar dari data tanpa diprogram secara eksplisit.

💡 Tahukah kamu? ML dan NLP ada di sekitarmu setiap hari — dari Google Assistant hingga sistem rekomendasi Netflix, semuanya menggunakan teknologi ini!

Jenis-Jenis Machine Learning

Machine learning dibagi menjadi 4 jenis utama berdasarkan cara mesin belajar dari data.

👨‍🏫
Supervised Learning
Pembelajaran Terbimbing

Model dilatih menggunakan data yang sudah memiliki label/jawaban. Seperti belajar di bawah bimbingan guru — data input dan output tersedia.

📧 Contoh: Filter spam email, deteksi penyakit dari foto rontgen, prediksi harga rumah
🔍
Unsupervised Learning
Pembelajaran Mandiri

Model belajar dari data tanpa label. Mesin menemukan pola dan struktur tersembunyi secara mandiri dalam data yang tidak berlabel.

🛒 Contoh: Segmentasi pelanggan, pengelompokan dokumen, deteksi anomali
🔀
Semi-Supervised Learning
Pembelajaran Semi-Terbimbing

Kombinasi data berlabel (sedikit) dan data tidak berlabel (banyak). Efisien ketika pelabelan data membutuhkan biaya dan waktu yang tinggi.

🏥 Contoh: Klasifikasi gambar medis, pengenalan ucapan, analisis teks
🎮
Reinforcement Learning
Pembelajaran Penguatan

Model belajar melalui interaksi dengan lingkungan menggunakan sistem reward dan punishment. Mesin mencari tindakan terbaik untuk memaksimalkan reward.

♟️ Contoh: AlphaGo, robot navigasi, kendali otomatis kendaraan
Algoritma

Algoritma Machine Learning

Pilih algoritma untuk mempelajari cara kerjanya secara interaktif.

Supervised Learning
Decision Tree

Algoritma yang membuat keputusan seperti pohon bercabang. Setiap node merepresentasikan pertanyaan/kondisi, dan setiap cabang adalah jawaban, hingga mencapai kesimpulan akhir di daun.

Contoh Penggunaan

  • Sistem rekomendasi produk berdasarkan preferensi
  • Klasifikasi genre film atau musik
  • Diagnosa penyakit berdasarkan gejala
  • Penilaian kredit nasabah bank

Supervised Learning
K-Nearest Neighbour

Algoritma yang mengklasifikasikan data berdasarkan kemiripan dengan K data tetangga terdekatnya. Prinsipnya: "Katakan padaku siapa tetanggamu, aku tahu siapa kamu."

Misalnya K=3, maka data baru akan diklasifikasikan berdasarkan mayoritas dari 3 data terdekatnya.

Contoh Penggunaan

  • Sistem rekomendasi film/buku serupa
  • Klasifikasi gambar (apel vs cupcake)
  • Deteksi penipuan transaksi
  • Pengenalan tulisan tangan
Visualisasi: Klasifikasi Titik Baru (K=3)

Supervised Learning · Regresi
Regresi Linier

Algoritma untuk memprediksi nilai numerik berkelanjutan. Mencari garis lurus terbaik (y = mx + b) yang mewakili hubungan antara variabel input dan output.

Contoh Penggunaan

  • Prediksi nilai ujian berdasarkan jam belajar
  • Prediksi harga rumah berdasarkan luas
  • Estimasi penjualan berdasarkan iklan
  • Prediksi cuaca (suhu)
Visualisasi: Jam Belajar vs Nilai
Unsupervised Learning · Clustering
K-Means

Algoritma yang mengelompokkan data ke dalam K kelompok (cluster) berdasarkan kemiripan karakteristik. Data dalam satu cluster memiliki karakteristik yang mirip satu sama lain.

Contoh Penggunaan

  • Segmentasi pelanggan berdasarkan perilaku belanja
  • Pengelompokan berita berdasarkan topik
  • Kompresi gambar digital
  • Analisis data gen bioinformatika
Visualisasi: Sebelum & Sesudah K-Means (K=3)

Unsupervised Learning
Association Rule

Algoritma untuk menemukan pola hubungan antara item dalam data. Contoh terkenal: "Market Basket Analysis" — menemukan produk yang sering dibeli bersamaan.

Contoh Penggunaan

  • Rekomendasi produk toko online ("Orang juga beli...")
  • Analisis pola pembelian supermarket
  • Rekomendasi playlist musik
  • Analisis pola klik website

Glosarium

Kumpulan istilah penting dalam Machine Learning dan Natural Language Processing.

Machine Learning (ML)
Cabang ilmu komputer yang memungkinkan sistem belajar dan berkembang dari pengalaman tanpa diprogram secara eksplisit.
Natural Language Processing (NLP)
Cabang AI yang berfokus pada interaksi antara komputer dan bahasa manusia, memungkinkan komputer memahami dan menghasilkan teks.
Supervised Learning
Jenis pembelajaran mesin menggunakan data yang sudah memiliki label/jawaban sebagai panduan pelatihan model.
Unsupervised Learning
Jenis pembelajaran mesin dari data tanpa label, mesin menemukan pola tersembunyi secara mandiri.
Reinforcement Learning
Metode pembelajaran dengan sistem reward dan punishment, mesin belajar dari interaksi dengan lingkungannya.
Training Data
Kumpulan data yang digunakan untuk melatih model machine learning agar dapat membuat prediksi yang akurat.
Decision Tree
Algoritma ML yang menggunakan struktur pohon keputusan untuk mengklasifikasikan data atau membuat prediksi.
K-Nearest Neighbour (KNN)
Algoritma klasifikasi yang menentukan kelas data baru berdasarkan mayoritas K data terdekat di sekitarnya.
K-Means
Algoritma clustering yang mengelompokkan data ke dalam K kelompok berdasarkan kemiripan karakteristik.
Tokenisasi
Proses memecah teks menjadi unit-unit kecil (token) seperti kata atau frasa dalam NLP.
Stemming
Proses menghapus imbuhan dari kata sehingga menghasilkan kata dasar (stem) dalam pemrosesan bahasa.
POS Tagging
Part of Speech Tagging — pemberian label gramatikal pada setiap kata (kata benda, kata kerja, kata sifat, dll).
Named Entity Recognition (NER)
Teknik NLP untuk mengenali dan mengklasifikasikan entitas dalam teks seperti nama orang, organisasi, dan lokasi.
Analisis Sentimen
Teknik NLP untuk mendeteksi dan mengklasifikasikan emosi/opini dalam teks sebagai positif, negatif, atau netral.
Model
Hasil akhir dari proses pelatihan machine learning yang dapat digunakan untuk membuat prediksi pada data baru.
Algoritma
Kumpulan instruksi matematis yang digunakan untuk memproses data dan menghasilkan output dalam machine learning.
Cluster
Kelompok data yang memiliki karakteristik serupa dalam algoritma clustering seperti K-Means.
Kecerdasan Artifisial (KA)
Simulasi kecerdasan manusia oleh mesin, terutama sistem komputer. Machine Learning adalah salah satu cabangnya.
